Bitacora de desarrollo

12 de abril de 2026

Inicio del proyecto

Decisiones de diseño: Se definio una estructura simple con HTML en la raiz y carpetas separadas para CSS, JS e imagenes.

Dificultades: El primer desafio fue mantener las paginas consistentes sin repetir demasiado codigo.

Cambios: Se creo la portada, las paginas individuales y la seccion de bitacora.

12 de abril de 2026

Interactividad inicial

Decisiones de diseño: Se agregaron botones para mostrar mensajes, cambiar el estilo de la foto y desplegar informacion extra.

Dificultades: Los scripts de cada integrante repetian la misma logica.

Cambios: Se mantuvo la funcionalidad y se identifico la necesidad de centralizar componentes.

28 de abril de 2026

Refactor reutilizable

Decisiones de diseño: Se separaron los datos del renderizado para que el sitio pueda crecer sin copiar bloques de HTML.

Dificultades: Habia textos mal codificados y duplicacion entre paginas.

Cambios: Se incorporaron js/data.js y js/app.js como base comun para la portada, integrantes y bitacora.
